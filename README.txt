1) Для использования бота, необходимо поместить в файл TgToken.txt токен бота, полученный у @BotFather.
1) Бот предназначен для уведомления о скачках на криптобирже бинанс(Futures, Spot).
2) Все настройки сохраняются в файле CONFIG.txt.
    a) Базовые значения(frequencyRefrashBaseValue) - частота обновления базовых значений в минутах (целочисленный формат).
    b) Базовые значения(minChangeABS) - частота отклонения актуальных данных от базовых по модулю, в процентах (целочисленный формат).
    c) Актуальные данные(frequencyRefrashRealValue) - частота обновления актуальных значений в секундах/минутах (числа с плавающей точкой и приставкой s/m для указания секунд/минут соответственно).
3) При возникновении скачка, бот отправляет уведомление в телеграм пользователя вида "Цена COINNAME spot/futures упала/выросла на x.xxxx% Первоначальная цена:x.xxxxxxxx Цена на текущий момент :x.xxxxxxxx".
4) Бот предназначен для локального использования, функционала рассылки нескольким пользователям не имеется.
